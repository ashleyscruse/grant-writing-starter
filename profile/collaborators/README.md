# Collaborators

The people you work with or plan to work with. One folder per person.

**To add someone:** copy the `_TEMPLATE/` folder, rename it for the person (e.g. `taylor-james/`), fill in `profile.md`, and drop their CV, biosketch, or collaboration letter in the same folder.

```
collaborators/
  taylor-james/
    profile.md        expertise, relationship (current or prospective)
    cv.pdf
    biosketch.pdf
```

When a grant needs expertise you don't have, the AI can look here to suggest team members, and it drafts personnel text from the real documents.
