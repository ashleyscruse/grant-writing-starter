# Collaborator Profile

One folder per key collaborator (current or prospective). These help Claude suggest team members when a grant requires expertise you don't have, and let it draft from the real record.

**To add a collaborator:** copy this whole `_TEMPLATE` folder to a folder named for the person (e.g. `taylor-james/`), fill in the `profile.md`, and drop their CV, biosketch, or collaboration letter into the same folder. Sage can fill the profile with you.
