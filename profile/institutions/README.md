# Institutions

Your institution(s). One folder per institution. Type and resources drive eligibility (HBCU, MSI, PUI, R1) and the Facilities section.

**To add one:** copy the `_TEMPLATE/` folder, rename it for the institution (e.g. `morehouse/`), fill in `profile.md`, and drop its facilities-and-resources document and any letters of support in the same folder.

```
institutions/
  morehouse/
    profile.md        type, resources, affiliations
    facilities.pdf
    letter-of-support.pdf
```
