# Institution Profile

Your institution type and resources matter for grant eligibility and framing. Some programs specifically fund HBCUs, MSIs, or PUIs. Others require certain infrastructure. This document helps Claude assess what you can and can't apply for.

One folder per institution (make more than one if you hold appointments at multiple places).

**To add an institution:** copy this whole `_TEMPLATE` folder to a folder named for the institution (e.g. `morehouse/`), fill in the `profile.md`, and drop its facilities and resources document or letters of support into the same folder. Sage can fill the profile with you.
