# Morehouse Supercomputing Facility (MSCF) pack

A ready institution profile and boilerplate so faculty can name and use MSCF compute in their proposals, and so the AI grounds Facilities, DMP, and feasibility language in real resources instead of guessing.

**This whole folder is the MSCF in-kind pack. Delete it for the generic / commercial release of the starter.** The core skills read `profile/institutions/` automatically, so nothing else references MSCF.

> Status: DRAFT. The reference content (allocation framing, exact specs, contacts, classification, programs) still needs refinement before anyone relies on it in a submission. `[REFINE]` marks the spots.

## Files

- `profile.md`: the institution profile the skills read for eligibility and capacity.
- `facilities-and-resources.md`: adaptable Facilities, Equipment & Other Resources language.
- `data-management.md`: the storage/compute half of a Data Management Plan.
- `working-with-mscf.md`: the human process: requesting support, an allocation, compute estimates, lead times, contacts.
- `letters-when-and-how.md`: when a letter is allowed and the NSF Letter of Collaboration format.
