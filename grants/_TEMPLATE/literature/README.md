# Literature

Drop the source material for this proposal here: papers (PDF), notes, key references, the solicitation's cited work, and your own prior results.

The AI reads this folder when it evaluates the opportunity and when it drafts, so it grounds claims and citations in **real sources you provided** instead of guessing. This is how you keep it from hallucinating references.

Rules:
- Put a source here before you ask the AI to cite it.
- If a claim is not supported by something in this folder or in your `profile/`, the AI should flag it for you to confirm, not invent a citation.
- Filenames are up to you. Author and year is a good habit, for example `Chen-2024-tnbc-methods.pdf`.
