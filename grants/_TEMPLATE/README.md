# (grant template)

Copy this whole folder to start a new grant, or let the `grant-advisor` skill create it for you. Rename it for the opportunity (no spaces), e.g. `nsf-hbcu-eir/`. Then:

1. Put the opportunity in `solicitation.md` (text or a link).
2. Drop any papers and references into `literature/` so the AI cites real sources, not guesses.
3. Run the `grant-advisor` skill to fill `solicitation.md` and `evaluation.md` and decide Pursue / Considering / Don't Pursue (it also logs the tracker).
4. If you decide to pursue, fill in `team.md` (who is on this proposal).
5. Draft in `proposal/`, using the render / feedback / revise loop. Numbers go in `budget.xlsx`.
