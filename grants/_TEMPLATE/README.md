# (grant template)

Copy this whole folder to start a new grant. Rename it for the opportunity (no spaces), e.g. `NSF-HBCU-EiR/`. Then:

1. Put the opportunity in `solicitation.md` (text or a link).
2. Run the grant-analyzer skill to fill `evaluation.md` and decide go / no-go.
3. If you're a go, fill in `team.md` (who's on this proposal).
4. Draft in `proposal/`, using the render / feedback / revise loop.
