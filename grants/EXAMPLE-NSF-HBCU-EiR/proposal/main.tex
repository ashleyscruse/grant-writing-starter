\section*{Project Description}

Triple-negative breast cancer (TNBC) disproportionately affects Black women, yet
existing genomic databases underrepresent these populations, leaving drug targets
understudied. This project develops ancestry-aware computational methods to identify
candidate drug targets in TNBC. The work is organized around three objectives.

\paragraph{Objective 1.} Build a single-cell atlas of TNBC tumor microenvironments in
African American patients.

\paragraph{Objective 2.} Develop a network-based drug-repurposing pipeline that maps
pathway dysregulation to existing compounds.

\paragraph{Objective 3.} Release an open-source toolkit for ancestry-aware genomic analysis.
